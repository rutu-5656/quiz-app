# 🚀 Quiz App (Full Stack)

A full stack quiz application built using React + FastAPI.

## 🔥 Features
- User Authentication (Login/Register)
- Practice Mode
- Compete Mode
- Score Tracking & History
- Admin Panel (Add/Delete Questions)

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: FastAPI (Python)
- Database: SQLite

## ⚙️ How to Run

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
